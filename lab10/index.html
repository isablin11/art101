Lab 10 - JavaScript for the Web

Challenge

The challenge of this lab was to experiment with DOM manipulation. With the main goal of this lab being to create a button that, when clicked, generated random "conversation". In this case We used a little bit of the "Lorem Ipsum" paragraph to generate the random conversation.

Problems

One of the problems we ran into with this lab was getting the button to actually work when clicked. In the beginning, we fund that when clicking it, nothing happened. Thats when we found out that we hadn't actually put our event lister callback and output appendage INSIDE the clicker button function. In the instructions for the lab it looks like the function just ends, but steps 4 and 5 actually needed to be put inside the function made in step 3.
